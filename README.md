I used webscraping to scrape BBC news website to output all news headers as news reports

## Settings
Also keep in mind, you can change the amount of news headers that will be printed after running the script. To do that find the 'scraping_range' variable in the code and change it to the amount of news you want to scrape.

## Features
- Fetches top 50 headlines from BBC News
- Extracts article links
- Identifies live broadcasts
- Handles missing links gracefully
- Includes user-agent header for request compatibility

## How to run
```bash
python news_scraper.py
```
!!! Make sure you have python installed on your computer and use CMD (Command Prompt) and run the bash script. !!!
